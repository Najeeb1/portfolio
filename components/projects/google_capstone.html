Google Capstone Project

Completed: Sep 2024
Technologies: Excel, R, Python, Pandas, BigQuery, Tableau

Project Overview

This project, which is part of the Google Data Analytics Professional Certificate program, demonstrates proficiency in data analysis using a range of tools and technologies. The project involves performing Exploratory Data Analysis (EDA) on a large dataset, deriving insights, and visualizing these insights effectively.

Key Features

  • Data cleaning and transformation using Python and Pandas
  • Exploratory Data Analysis (EDA) using R and Excel
  • Data visualization through Tableau
  • SQL queries for data extraction using BigQuery
  • Analyzing trends, distributions, and correlations in the dataset
  • Visualizing key findings through charts and graphs
  • Providing actionable insights for business decision-making

Technical Implementation

The project was carried out using Excel, R, Python, Pandas, BigQuery, and Tableau. Data cleaning, transformation, and analysis were performed using Python and Pandas, while EDA was carried out with R and Excel. Key insights were visualized using Tableau, and SQL was used for extracting data from BigQuery. Key technical features include:

  • Data wrangling and cleaning using Python and Pandas
  • Exploratory Data Analysis (EDA) with R and Excel
  • Data visualization with Tableau
  • Querying large datasets with BigQuery
  • Creating reports and dashboards to present insights effectively

Project Challenges

Some of the key challenges faced during the development process included:

  • Handling and cleaning large and messy datasets
  • Ensuring efficient queries for extracting data from BigQuery
  • Finding the right data visualizations to communicate insights clearly
  • Integrating various data sources and ensuring data consistency

Future Improvements

Future improvements and enhancements for the project include:

  • Adding more complex visualizations and interactive dashboards in Tableau
  • Optimizing SQL queries for better performance with larger datasets
  • Integrating additional data sources for a more comprehensive analysis
