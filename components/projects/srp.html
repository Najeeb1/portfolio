Hierarchical Time Series Forecasting (SRP)

Completed: 2025
Technologies: DeepVAR, Hier-E2E, Time Series Forecasting, CRPS, Python

Project Overview

This project focuses on Hierarchical Time Series Forecasting, a crucial task in various fields such as sales monitoring, electricity consumption, and traffic management. Traditional forecasting models often deal with deterministic predictions, but uncertainty quantification is needed to provide more reliable insights. The current state-of-the-art model, Hier-E2E, uses a multivariate forecaster for generating probabilistic forecasts, with reconciliation done through a projection step to maintain coherence across hierarchical variables. We propose a novel approach called HiT that eliminates the projection step for reconciliation, thus simplifying the model while preserving forecast accuracy. The HiT model is trained by feeding only the bottom-level time series to the DeepVAR model, and higher-level forecasts are obtained by summing the predictions from the relevant bottom levels. A new loss function is introduced to inherently handle hierarchical constraints, improving the overall forecast quality.

Key Features

  • Hierarchical time series forecasting for fields like sales, electricity consumption, and traffic management
  • Introduction of HiT, a novel model for eliminating the projection step for reconciliation
  • Training the HiT model using DeepVAR for bottom-level time series forecasting
  • New loss function designed to manage hierarchical constraints effectively
  • Comparison of HiT's performance against baseline models using real-world datasets
  • Demonstration of HiT's better CRPS score compared to the state-of-the-art Hier-E2E model

Technical Implementation

The HiT model utilizes DeepVAR to forecast the bottom-level time series data, where higher-level forecasts are generated by summing the forecasts from appropriate lower levels. This approach eliminates the need for the projection-based reconciliation in traditional models like Hier-E2E. To train the HiT model, we introduced a new loss function that inherently handles the hierarchical nature of the time series data. The model is evaluated on three real-world datasets, with performance compared to baseline models based on CRPS (Continuous Ranked Probability Score). Key technical features include:

  • Building the HiT model with DeepVAR for forecasting time series data
  • Eliminating projection-based reconciliation for simpler hierarchical forecasting
  • Designing a new loss function to handle hierarchical constraints effectively
  • Evaluating model performance using CRPS on real-world datasets
  • Benchmarking HiT against Hier-E2E and other baseline models

Project Challenges

Some of the challenges encountered during the development process included:

  • Ensuring accurate hierarchical predictions without relying on the projection step
  • Designing a loss function that can naturally enforce hierarchical constraints
  • Dealing with the inherent complexity of time series forecasting across multiple levels
  • Benchmarking the HiT model against traditional, more complex models
